# Start Guide

Start Guide creates the first real workflow without loading demo data. It is
optional and opens only when the user selects **Start Guide** in the sidebar.

The three steps ask for:

1. organization or team and project/theme;
2. the first work domain and what is currently stuck;
3. the boundary where AI must stop for human judgment.

Kazane then creates exactly two linked records:

- a Context Card containing the origin, purpose, constraints, and unresolved issue;
- an Inbox Work Item linked to that Context Card and project.

The Work Item is assigned to AI番頭 for initial organization, marked for the
morning dashboard, and uses the selected domain template. Start Guide supports
Japanese, English, and Chinese UI copy. Business answers remain in the language
entered by the user.
