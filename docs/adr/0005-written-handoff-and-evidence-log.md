# ADR 0005: Written handoff and evidence log

- Status: Accepted
- Date: 2026-06-26

## Context

Ephemeral chat is insufficient for human-AI organizations. Work must be traceable across agents and humans.

## Decision

Require durable Handoff Notes and Evidence Logs for meaningful work transitions and audit-relevant outputs.

## Consequences

This increases traceability. It may add friction unless UI and automation make handoffs easy.

## RDE check

- Preserved: written handoff as core operating model.
- Transformed: work completion becomes traceable work transition.
- Supplemented: evidence linkage.
- Unresolved: minimum evidence requirements by work type.
- Deviation risk: handoff may become verbose ritual.
- Next update: define Handoff Note schema.
